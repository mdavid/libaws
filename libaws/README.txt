  ----------------------------
  28msec, Inc. - libaws
  ----------------------------

  libaws is a C++ library that allows to communicate with Amazon Web Services (e.g. S3 and SQS).

  Documentation
  -------------

  The documentation is included in the doc directory. To generate
  the documentation execute "make doc" in a configured CMake build
  directory.

  Installation
  ------------

  Please see the documentation on the web site or the doc/html folder
  of the package.
